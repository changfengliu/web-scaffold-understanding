# CSS 预处理

绝大多数 CSS 预处理器会增加一些原生 CSS 不具备的特性，例如混合，嵌套选择器，继承选择器等。

这些特性让 CSS 的结构更加具有可读性和易于维护。

## 流行的处理器:

- [Sass](http://sass-lang.com/)
- [Less](http://lesscss.org/)
- [Stylus](http://stylus-lang.com/)
- [PostCSS](http://postcss.org/)
